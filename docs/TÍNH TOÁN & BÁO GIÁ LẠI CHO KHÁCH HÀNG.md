## 3. Công thức Tính toán Cơ bản
**Lưu ý:** Đơn vị báo giá thường là **cm**, nên khi tính toán nhớ quy đổi ra đơn vị **mét** (nếu cần).

### Tính m3 (Khối)
$$m^3 = \frac{\text{dài} \times \text{rộng} \times \text{cao}}{1.000.000} \times \text{số viên}$$

* **Ví dụ:** Xuất 500 viên đá basalt 10x10x8 cm
    * $m^3 = \frac{10 \times 10 \times 8}{1.000.000} \times 500 = 0.4$

### Tính m2 (Diện tích)
$$m^2 = \frac{\text{dài} \times \text{rộng}}{10.000} \times \text{số viên}$$

* **Ví dụ:** Xuất 180 viên đá basalt 40x20x3 cm
    * $m^2 = \frac{40 \times 20}{10.000} \times 180 = 14.4$

### Tính Tấn (Trọng lượng)
$$\text{Tấn} = m^3 \times \text{Trọng lượng riêng (TLR)}$$

**Trọng lượng riêng tham khảo:**
* **Đá granite, bluestone:** $m^3 \times 2.7$. (Dark grey granite: $\times 2.9$)
* **Đá bazan:** $m^3 \times 2.65$ (Gia công đặc biệt có thể là 2.7).
    * Absolute bazan: 2.95 tấn.
    * Black bazan: 2.65 hoặc 2.7 tấn.
* **Đá tổ ong:** $m^3 \times 2.2$.

## 4. Công thức Quy đổi Giá

**Ví dụ quy cách:** 10x25x100cm.
* **Số viên** = $m^3$ / dài / rộng / cao.
* **Lưu ý:** Dài, Rộng, Cao phải đổi sẵn ra mét.

### Từ Giá Viên
* Giá $m^2$ = Giá Viên : 0.1 : 0.25
* Giá $m^3$ = Giá Viên : 0.1 : 0.25 : 1
* Giá Tấn = Giá Viên : D : R : C : TLR : HS ốp đáy

### Tính ra Giá Viên
* Giá Viên = Giá $ml$ x 0.1
* Giá Viên = Giá $m^2$ x 0.1 x 0.25
* Giá Viên = Giá $m^3$ x 0.1 x 0.25 x 1

### Quy đổi giữa m2, m3 và Tấn
* Giá $m^2$ = Giá $m^3$ x Chiều cao
* Giá $m^2$ = Giá Tấn x C x TLR x HS ốp đáy
* Giá $m^3$ = Giá $m^2$ : Chiều cao
* Giá $m^3$ = Giá Tấn x TLR x HS ốp đáy
* Giá Tấn = Giá $m^3$ : TLR : HS ốp đáy
* Giá Tấn = Giá $m^2$ : C : TLR : HS ốp đáy
